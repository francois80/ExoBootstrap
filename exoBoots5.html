Exercice 5 bootstrap

Pour utiliser efficacement Bootstrap, vous devez déjà être convaincu de son utilité, vous devez aussi savoir l'installer.
Depuis la refonte de Github, la plateforme utilise la version 4 du framework. L’opportunité de faire un rapide aperçu de leur utilisation de Bootstrap.
L’utilisation de Bootstrap se rapporte souvent à des sites de « petite importance » comme votre site web personnel ou un blog. Cependant, il est toujours agréable de voir que l’utilisation du framework peut s’appliquer à de plus gros projets, voir dans le cas précis à de très grosses structures telles que Github.
Pour la majorité d’entre nous, installer Bootstrap se résume à télécharger les fichiers sources directement par le site officiel getbootstrap.com. Bien entendu, il existe des manières plus directes et « professionnelles » sans passer par le site. C’est le cas avec Nodejs et la commande npm.
Par l’intermédiaire de commande dans le terminal de votre ordinateur, vous pourrez télécharger et installer directement des packages tels que Bootstrap 4 sans passer par le site officiel de Bootstrap.
Les font-size reponsive sont sans conteste la plus grosse addition pour cette mise à jour 4.3. L’équipe core explique qu’ils ont définit un calcule automatique pour la font-size basé sur les dimensions de l’écran final de l’utilisateur et/ou sur les viewport du navigateur. Ce calcule permet de rendre les font-size responsive dans tout les cas de figure possibles.
